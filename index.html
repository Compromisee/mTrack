Setup

Live in four steps.

Everything from zero to your local movie tracker.

1

Get a Free TMDB API Key

Create an account at themoviedb.org — Settings → API → Developer. Copy your v3 auth key.

2

Clone & Install

$ git clone https://github.com/Compromisee/mTrack.git
$ cd mTrack/movie-tracker
$ npm install
3

Add Your API Key to data.json

{ "tmdb_api_key": "your_key_here" }
4

Start & Browse

$ node server.js
ready → http://localhost:3000
Never commit data.json with your API key to a public repository. Add it to .gitignore before pushing.
